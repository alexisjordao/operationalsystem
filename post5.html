Conceitos básicos
● IDE (Integrated Development Environment)
○ Ambiente de desenvolvimento (e.g eclipse)
● Library (lib)
○ Código que pode ser chamado/linked a partir de um programa (DLL ou estático)
● API (Application Programming Interface)
○ Interface de acesso uma biblioteca (uma lib pode ser atualizada e a API continuar sendo a mesma)
● SDK (Software Development Kit)
○ Conjunto de programas integrados para o desenvolvimento. Pode conter libs, IDE, arquivos de ajuda (e.g SDK Java, TI)
● Tookit
○ Biblioteca associada a ferramentas de desenvolvimento, porém, distribuida atraveś de plugins
● Framework
○ Plataforma de desenvolvimento (amplo)

Atualização do Sistema [Debian Jessie]
Download 
wget https://rcn-ee.com/rootfs/2016-02-11/elinux/debian-8.3-console-armhf-2016-02-11.tar.xz
Unpack
tar xf debian-8.3-console-armhf-2016-02-11.tar.xz
cd debian-8.3-console-armhf-2016-02-11
Identificando o microSD
sudo ./setup_sdcard.sh --probe-mmc
sudo ./setup_sdcard.sh --mmc /dev/mmcblk0 --dtb beaglebone

Acessando o sistema
ssh debian@192.168.7.2
Senha: temppwd

Instalando alguns pacotes [Máquina Host]
sudo apt-get install sshpass
